Description globale : **Une exception à la restriction représente un cas particulier qui spécialise la restriction signalée dans une zone de circulation restreinte ; par exemple, elle peut décrire des autorisations particulières appliquées à un type de véhicule spécifique**.  
version : 0.1.0
